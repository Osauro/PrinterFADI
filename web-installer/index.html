POS FADI

Sistema de Punto de Venta Profesional

Versión 1.0.0

Características Principales

  • Impresión Térmica - Compatible con impresoras POS 58mm/80mm
  • Códigos QR - Generación automática para recibos
  • Configuración Visual - Interface web para configurar impresora
  • Múltiples Formatos - Ventas, boletas, transferencias
  • Base de Datos Remota - Conecta con servidor fadi.com.bo
  • Servidor Embebido - PHP integrado, sin dependencias
  • Auto-actualización - Descarga automática desde GitHub
Tecnologías
Laravel 10
PHP 8.2
Bootstrap 5
MySQL

Instalación Automática

Requisitos del Sistema
  • Sistema: Windows 10/11 (64-bit)
  • Memoria: 2GB RAM mínimo
  • Espacio: 500MB libres
  • Red: Conexión a internet para instalación
  • Impresora: POS térmica (opcional)
Descargar Instalador
(~15MB)
Instalación automática: El instalador descargará el código fuente desde GitHub, instalará todas las dependencias y configurará el servidor local automáticamente.
